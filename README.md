## ⚙️ How I Work

```
request  →  validate at the edge  →  service boundary  →  business logic  →  audited write
                 ↑                                                              ↓
            reject early                                                  log + respond
```

I'd rather catch a bad request at the door than debug it three services deep. That mindset shapes everything from auth middleware to error messages — fail fast, fail clearly, leave a trail.
