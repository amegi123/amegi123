## 🧭 The Blueprint

> *"Architecting scalable enterprise ecosystems through Microservices and Secure Logic."*

I'm a Full-Stack Software Engineer who treats backend architecture like infrastructure, not glue code — **secure by default, observable by design, and built to scale past the demo.** I bridge distributed systems with interfaces people actually enjoy using.

<table>
<tr>
<td width="50%" valign="top">

**🛡️ Security Layer**
Multi-tenant auth flows built on **Better Auth** + **JWT**, designed so permissions fail closed, not open.

**🏗️ Service Mesh**
**NestJS** microservices, containerized with **Docker**, fronted by **Nginx** — each service does one job well.

</td>
<td width="50%" valign="top">

**🎨 Interface Layer**
Accessible, premium UI with **Shadcn UI**, **Radix**, and **Tailwind CSS** — components that respect keyboard and screen reader users.

**🏆 Proof of Work**
2nd Place — Safaricom Capstone (FuelFinder), built under real competition constraints.

</td>
</tr>
</table>

## ⚙️ How I Work

```
request  →  validate at the edge  →  service boundary  →  business logic  →  audited write
                 ↑                                                              ↓
            reject early                                                  log + respond
```

I'd rather catch a bad request at the door than debug it three services deep. That mindset shapes everything from auth middleware to error messages — fail fast, fail clearly, leave a trail.
